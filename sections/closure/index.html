How To Find the Closure of a Set of Attributes

The closure of a set of attributes X is the set of those attributes that can be functionally determined from X. The closure of X is denoted as X+.

When given a closure problem, you’ll have a set of functional dependencies over which to compute the closure and the set X for which to find the closure. A functional dependency A1, A2, …, An -> B in a table holds if two records that have the same value of attributes A1, A2, …, An also have the same value for attribute B. The closure of X is the set of all attributes such that two records that have the same value of X also have the same value of X+.
